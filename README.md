# BBB Tools
Simple tools & functions for the online video conference software "BigBlueButton".

# How to download
Click "[releases](/releases)" on the right side of this page and download the file(s) you need.

# How to use
Right-click a presentation and choose "inspect element" or press `Ctrl+Shift+I`. Double-click on the underlined code or right-click and choose "Edit as HTML". Try to find the presentation URL and copy it. The format is (for example) the following one: `https://bbb.talpaworld.de/bigbluebutton/presentation/XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX-0000000000000/XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX-0000000000000/XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX-0000000000000/svg/000`. The URL starts with `https://` and ends with `/svg/123` (for example of course, the number can variy). Finally, paste the URL in the program box.
